# Action Agenda

## Quickstart

1. `bundle install`
2. `yarn install`
3. `rails db:create db:migrate`
4. `rails s` 
5. Obtain a copy of the 'commitments' CSV if not already present in `lib/data/seeds`
6. Run `rails import:commitments`
7. On the Commitments page, you should see a list of commitments.
